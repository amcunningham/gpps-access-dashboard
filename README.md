# GPPS Access Dashboard

Self-contained HTML dashboard analysing GP Patient Survey access satisfaction in England (2007–2025) against the timeline of major access policy interventions.

**Live dashboard:** enabled via GitHub Pages — see the repository's Pages URL.

## Contents

- `index.html` — complete dashboard (Chart.js via CDN, all data embedded inline). Open locally in any modern browser, or view on GitHub Pages.

## Structure

Seven tabs:

1. **Phase 1: Timeline** — overall experience, phone access, appointment experience, and GP workforce (FTE) vs satisfaction, with policy interventions annotated.
2. **Phase 2: Was There a Crisis?** — pre-Challenge Fund satisfaction levels.
3. **Phase 3: Did Interventions Work?** — pre- vs post-intervention comparison, extended access (BJGP Open 2022), waiting time distributions.
4. **Phase 4: Deprivation & Equity** — practice-level analysis linking 6,250 practices to IMD 2019 scores.
5. **Phase 5: Consultation Mode** — appointment mode mix, mode/satisfaction, F2F and workforce scatters.
6. **Narrative Synthesis** — written summary of findings.
7. **Data & Methods** — sources, questionnaire breaks, limitations.

## Data sources

GP Patient Survey (gp-patient.co.uk), NHS England Statistics, NHS Digital Appointments in General Practice, NHS England GP Workforce (wMDS), IMD 2019 (via Fingertips), BJGP Open 2022, University of Manchester / BJGP 2025.

All data is publicly available; no ethics approval required for secondary analysis.
